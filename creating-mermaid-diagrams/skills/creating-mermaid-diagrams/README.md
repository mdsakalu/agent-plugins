# Creating Mermaid Diagrams

A Claude Code skill for generating GitHub-compatible Mermaid diagrams with tested color palettes, local preview, and gist-based GitHub preview.

## Features

- All Mermaid diagram types with GitHub-compatible syntax
- 24 named color themes tested on GitHub light and dark modes
- Local SVG/PNG preview via mermaid CLI
- GitHub rendering preview via private gists
- Gist lifecycle management with user-confirmed cleanup
- Accessibility support (accTitle/accDescr)
